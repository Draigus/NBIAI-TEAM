## Technical Reference

### Paperclip Agent Schema -- Key Fields for PATCH /api/agents/:id

| Field | Type | Purpose |
|---|---|---|
| `capabilities` | string | Description text shown in UI |
| `title` | string | Job title |
| `reportsTo` | uuid | Manager agent ID |
| `desiredSkills` | string[] | Array of skill UUIDs |
| `adapterConfig` | jsonb | Instruction paths, adapter settings |
| `runtimeConfig` | jsonb | Heartbeat settings, wake-on-demand |
| `budgetMonthlyCents` | integer | Monthly budget in cents |
| `status` | string | idle, paused, pending_approval, terminated |

**NOT settable via PATCH:**
- `permissions` -- use PATCH /api/agents/:id/permissions
- `spentMonthlyCents` -- read-only
