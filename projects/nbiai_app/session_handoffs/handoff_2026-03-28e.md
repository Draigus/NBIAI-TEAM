### Sprint 6: Task Scheduler Trigger Scripts

**`queue/scripts/process-inbox.ps1`:**
- Scans `queue/inbox/` for the oldest task JSON
- Moves it to `queue/active/`, updates status and claimed_at
- Copies the full session prompt to clipboard
- Launches Claude Desktop if not already running
- Logs all actions to `process-inbox.log`

**`queue/scripts/post-results.ps1`:**
- Accepts `-TaskId`, `-Status`, `-OutputFile`, `-Notes` parameters
- Reads output from file and POSTs to `POST /api/v1/queue/results`
- Requires `NBIAI_AUTH_TOKEN` environment variable for auth
- Logs results to `post-results.log`

**`queue/scripts/README.md`:** Updated with full usage docs for both scripts

**`queue/schedules/nbiai-queue-processor.xml`:**
- Windows Task Scheduler XML config
- Runs every 15 minutes, weekdays 07:00-22:00
- Import: `schtasks /create /xml "...\nbiai-queue-processor.xml" /tn "NBIAI\Queue Processor"`

### Environment Setup (one-time)

1. Install PostgreSQL 16: `winget install PostgreSQL.PostgreSQL.16`
2. Create database:
   ```sql
   CREATE USER nbiai WITH PASSWORD '<password>';
   CREATE DATABASE nbiai OWNER nbiai;
   ```
3. `cd projects/nbiai_app/app && cp .env.example .env` -- fill in values
4. `npm install`
5. `cd client && npm install && cd ..`
6. `npm run db:migrate`
7. `npm run db:seed`
8. `npm run dev` (API on :3001)
9. `cd client && npm run dev` (frontend on :5173)
10. Smoke test: `http://localhost:3001/health`
