## Fixes Applied

### Screenshot Capture (Bug Report Modal)
- **Root cause 1**: CSP `script-src` didn't include `cdnjs.cloudflare.com`, blocking html2canvas CDN load
- **Root cause 2**: html2canvas 1.4.1 can't parse modern CSS `color()` function — throws during CSS parsing
- **Fix**: Added cdnjs.cloudflare.com to CSP, wrapped html2canvas call with error handling, added `onclone` to strip `color()` values from cloned DOM, 8s timeout
- **Files**: `dashboard-server/server.js:226`, `nbi_project_dashboard.html:9183-9220`

### P1: Task View Freeze (1,123 tasks)
- **Root cause**: All 1,123 tasks rendered as DOM nodes synchronously, even when collapsed
- **Fix**: Lazy rendering — collapsed children not rendered at all; populated on-demand when expanded via `toggleChildren()` and `toggleClientGroup()`. Added `getDepth()` helper.
- **Result**: 129ms render (was 30+ seconds), 0 DOM nodes initially (vs 1,123+)
- **Files**: `nbi_project_dashboard.html:4101-4106` (renderTaskRow), `4124-4180` (toggleChildren/toggleClientGroup/getDepth), `3258-3261` (client group lazy render)

### P1: Finance PUT Race Condition
- **Fix**: Added optimistic concurrency — GET /api/finance returns `version` (row id), PUT accepts `expectedVersion`, returns 409 Conflict if stale
- **Frontend**: Tracks `_financeVersion`, sends on save, auto-reloads on conflict with toast
- **Files**: `server.js:1517-1550`, `nbi_project_dashboard.html:5639,5690,5706-5720`

### P2: Pipeline NaN
- **Root cause**: `GROUP BY l.currency` created separate rows per currency (including NULL for empty stages)
- **Fix**: Removed `l.currency` from GROUP BY, aggregates across all currencies per stage
- **File**: `server.js:2502-2515`

### P2: People View Duplicates
- **Root cause**: Free-text assignee names ("Glen" vs "Glen Pryer") treated as different people
- **Fix**: Added `normaliseAssignees()` function that merges partial name matches (prefix-based). Added `isAssignedTo()` helper used in all people view filters. Server-side capacity route also updated with fuzzy name matching.
- **Files**: `nbi_project_dashboard.html:5257-5290` (normaliseAssignees, isAssignedTo), multiple `includes()` calls updated throughout people view. `server.js:4103-4115` (capacity route)

### P2: BUG-001 Account Lockout
- **Fix**: Lowered lockout threshold from 10 to 5 (flag at 3), added logging for lockout events, added `locked` and `minsLeft` to 429 response
- **File**: `server.js:100-103, 274-284`

### P2: BUG-003 XSS in Emails
- **Fix**: Applied `escHtml()` to `user.display_name` in password reset HTML email and expense reminder notification
- **File**: `server.js:413, 4219`

### Bonus: Finance Data Schema Crash
- **Root cause**: DB finance_data had partial schema (only `opex` + `employerCostPct`), missing `revenue`, `payroll`, `pipeline`, `targets`
- **Fix**: `getFinanceData()` now merges with `NBI_FINANCE_SEED` defaults for missing keys
- **File**: `nbi_project_dashboard.html:5677-5689`
