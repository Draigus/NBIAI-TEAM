# NBI AI Operations Architecture and Service Offering

## Key Content

NBI runs an LLM Operating System (Karpathy framework): Claude as kernel, NBI_Brain.md + 11 modules as persistent storage, 33-role agent team as domain-specific processing units, custom Claude Code skills as system calls, approval gates as permissions, model tier routing (Opus/Sonnet/Haiku) as resource allocation. Internal impact evidence: Goals engagement ($10K, 21 tasks, 3 output formats, 3-way alignment verified automatically) delivered by single consultant with AI backing what would normally need PM + analyst + document specialist. Competitive research pipeline: 315 price points across 8 titles with red-team validation scoring 93/100. Bug triage: 7-step mandatory pipeline, 387 tests, PM2 + Cloudflare Tunnel. Daily Command Centre aggregates 8 data sources. Headline: 7-person firm delivering at 3-4x team depth. Client service offering: Studio Brain (persistent institutional knowledge), role-specific knowledge bases, session continuity systems. The system writes to disk in real time so continuity is mechanical, not dependent on memory.
