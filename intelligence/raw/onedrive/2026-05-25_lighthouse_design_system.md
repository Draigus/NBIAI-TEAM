# Client-Specific Design System: Lighthouse Analytics Dashboard

## Key Content

NBI created a branded design system for Lighthouse Games' analytics dashboard. Magenta/blue/pink palette, dark-first, cinematic automotive theme. Primary: #A93DA1 (magenta), Secondary: #3152FF (deep blue), Accent: #E4347A (pink), #FF5528 (coral). Surface hierarchy: base #0F032B, card #1A0E30, elevated #251842, hover #30204F. 8-colour chart palette plus 7-step funnel gradient. Typography: Inter font family, sizes from 11px (overline) to 28px (h1), weights 400-700. This demonstrates NBI's capability to deliver client-branded analytics dashboards that are visually distinct from generic templates.

## Decisions / Insights

- NBI decided: client dashboards get bespoke design systems, not NBI's default palette
- NBI observed: dark-first design with cinema-grade colour depth creates premium feel for analytics tools
- NBI decided: funnel visualisations get their own dedicated gradient (7 steps) separate from chart colours
