# Rami Ismail's Levelling The Playing Field -- Milestone Framework for Indie Game Development

## Key Content (max 200 words)

Four-phase milestone framework designed for 6-10 person indie teams, published as part of Rami Ismail's "Levelling The Playing Field" resource. Phases: Research (Ideation + Prototyping), Pre-Production (Vertical Slice + Production Start), Production (Feature Complete/Alpha + Content Complete/Beta), Wrap-Up (Release Candidate + Release). Each sub-stage has explicit gate criteria, deliverables, and team composition guidance.

Critical transitions: Vertical Slice validates the production pipeline and proves the team can hit envisioned quality; Production Start locks core "verbs" (mechanics) and establishes the development timeline. Feature Complete means all mechanics functional with minimal content. Content Complete means game playable start-to-finish with tutorialisation and localisation ready.

Vertical Slice can consume up to one-quarter of total development time. Content Complete should be reached 8-12 weeks before launch (4 weeks absolute minimum). Framework explicitly recommends Notion, Trello, Codecks, Jira, or Asana for progress tracking.

## Decisions / Insights

- The Vertical Slice is not a demo; it is a pipeline validation exercise that determines whether production is viable
- Feature-to-content ratio varies by genre and must be discussed early -- RPGs are content-heavy, puzzle games are feature-heavy
- Buffer time is mandatory for time-bound projects; the framework warns against treating milestones as precise deadlines
- "Never submit unstable builds for certification; only submit when comfortable shipping"

## Context

Rami Ismail is a Dutch-Egyptian game developer (Vlambeer co-founder, Nuclear Throne, Ridiculous Fishing) who created LTPF as a free resource for indie developers worldwide. The framework represents distilled experience from shipping multiple titles with small teams and from consulting with hundreds of indie studios globally. It is the most widely referenced indie milestone framework in the English-speaking game development community.

## Applicability

Directly applicable to NBI's studio clients in the 6-30 person range. The framework needs scaling adaptations for 30-100 person teams (adding discipline-specific sub-milestones and cross-department coordination). The gate criteria between phases can be adopted verbatim as publisher milestone definitions. Particularly valuable for NBI's production consulting: provides a shared vocabulary and checkpoint system that studios and publishers can agree on.
