# Playsage Competitive Analysis: Feature Benchmark Matrix

## Key Content
Detailed feature comparison across five competitors: Steam Spy (free, Steam-only, 24h latency, owners only no revenue), Newzoo GPM (enterprise, 4 platforms, daily-monthly, full revenue/retention), Sensor Tower VGI (enterprise, 3 platforms, 12h latency, UA linkage), VG Insights (Steam-only, daily, accurate +/-15% for 84% of games), GameAnalytics (SDK-based, real-time, opt-in bias). Key market gaps identified: no tool unifies Steam+Epic+Switch+PlayStation+Xbox; sub-hour refresh absent; behavioural/psychographic segmentation surface-level; scenario modelling manual.

Ten Playsage USPs: (1) Sub-hour refresh vs 12-24h competitors. (2) True cross-store index (5 stores). (3) AI scenario sandbox with causal ML. (4) Behavioural lens from NBI psychographic survey of 75k gamers/year. (5) Live-ops alerts with NLP patch diff. (6) Transparent accuracy meter with MoE per datapoint. (7) Indie-first pricing (free under USD100k gross rev). (8) Private data calibration via secure vault ingest. (9) API-native design. (10) Integrated NBI consulting escalation.

## Decisions / Insights
- Glen decided: Playsage differentiates on sub-hour refresh, cross-store coverage, and transparent accuracy
- Glen decided: indie-first pricing model (free below USD100k gross revenue)
- Glen observed: no existing tool covers Steam, Epic, and all three console platforms in one index
- Glen observed: VG Insights achieves +/-15% accuracy for 84% of Steam games, setting the baseline Playsage must beat
