# AAA Agilefall Production Operating Guide

## Key Content
Comprehensive producer onboarding guide for hybrid Agile+stage-gate (Agilefall) studios. Structure: (1) Foundations: product vision, 3-5 player promises, non-goals, pillar/track/owner structure. (2) Hybrid shape: gates on top (Discovery, Pre-production, Production, Alpha, Beta, Gold) with sprints underneath. Pre-production exits with playable vertical slice. (3) Backlog hierarchy: Epics > Features > User Stories > Tasks with Definition of Ready (clear problem, acceptance criteria, dependencies, assets, team agreement) and Definition of Done (criteria met, code reviewed, assets integrated, tests pass, docs updated, build green). (4) Two-horizon planning: quarterly PI planning (8-12 weeks) plus sprint planning (1-2 weeks). (5) Weekly rhythm: daily standup 15min, backlog refinement 60-90min, sprint planning 2-4h, sprint review 60-120min, retrospective 60min.
