**Key points first**

1. *Memory* in ChatGPT = facts about you that survive across sessions.  
   *Context window* = how many tokens the model can see in one chat. They are separate.  

2. You cannot “hack” either limit, but you can choose plans and features that give you more room or store information outside the model.

---

### Grow the built-in Memory feature

- Go to **Settings → Personalization → Memory**.  
  Turn it on, then tell the bot “Remember that …” whenever something matters.  
  You can view, edit or delete items in the same menu, or just say “Forget that”. citeturn1search3

- Plus and Pro tiers keep memories longer and use them more aggressively. Free users now get a lightweight version that references only recent chats. citeturn2view0

- Roll-out is still blocked in the UK, EU and a few EEA countries, so if you live there the switch will be greyed out for now. citeturn0news19turn1news10

---

### Pick a larger context window

Current limits (tokens shown are input + output combined):

| Tier | Context window |
|------|----------------|
| Free | 8 k |
| Plus / Team | 32 k |
| Pro / Enterprise | 128 k |

Upgrading is the only way to raise this ceiling inside ChatGPT. citeturn4search6

---

### Workarounds when the window is still too small

- **Summarise and re-paste**: Ask ChatGPT to make concise summaries of earlier parts of the dialogue, then continue with those summaries instead of the full text.

- **Projects or Custom GPTs**: Store your background documents in a Project or in a custom GPT, then reference them when needed rather than keeping them in the active chat.

- **External “memory scroll” file**: Keep a running doc (Markdown, Notion, etc.) with facts you want the model to recall. Attach or copy the relevant chunk into new chats. This is how many power users simulate persistent memory outside the official feature.
