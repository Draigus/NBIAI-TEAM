# 3) Retention_Curve
Column A = Day_Age 1..365  
Column B = Retention (decimal)

Use a simple, stable piecewise power model that hits your anchors exactly.

Anchors (pull from Inputs_Overview by scenario):
- B1 (for day 1) = D1%
- B7 (for day 7) = D7%
- B30 (for day 30) = D30%

Fit exponents between anchor pairs:
- **Segment 1** age 1‚Üí7: r(d) = A1*d^b1 with b1 = LN(r7/r1)/LN(7/1) and A1 = r1  
- **Segment 2** age 7‚Üí30: r(d) = A2*d^b2 with b2 = LN(r30/r7)/LN(30/7) and A2 chosen so r(7) matches r7  
- **Segment 3** age >30: continue with b2, or cap with a gentle tail: r(d) = r30*(d/30)^b2

Excel cells (example, assuming r1 in D1, r7 in D2, r30 in D3):
- `b1` in E1: `=LN(D2/D1)/LN(7/1)`  
- `b2` in E2: `=LN(D3/D2)/LN(30/7)`  
- In B2 (day=1): `=D1/100`  
- In B3..B8 (days 2..7): `=($D$1/100)*A2^$E$1`  
- In B9..B31 (days 8..30): `=($D$2/100)*(A9/7)^$E$2`  
- In B32..B366: `=($D$3/100)*(A32/30)^$E$2`  
Then wrap each with `MAX(0,MIN(1, ... ))` and enforce monotonic by `=MIN(previous_cell, this_cell)`.

# 6) Monthly_Summary
First add a Month key on Daily_Forecast:
- Month = `=TEXT(Date,"yyyy-mm")`

Then aggregate with SUMIFS/AVERAGEIFS:
- Installs = `SUMIFS(Installs_Total, Month, this_month)`  
- Avg DAU = `AVERAGEIFS(DAU, Month, this_month)`  
- Gross Revenue (GBP) = `SUMIFS(Gross_Revenue_Day, Month, this_month)`  
- Net Revenue (GBP) = `SUMIFS(Net_Revenue_Day, Month, this_month)`  
- UA Spend (GBP) = `SUMIFS(UA_Spend_Day, Month, this_month)`  
- **MAU (approx)**: good simple proxy = `SUMIFS(DAU, Month, this_month)*0.85`

- **ARPDAU (GBP)** = `AVERAGEIFS(ARPDAU, Month, this_month)`  
- **ARPU (GBP)** = `Gross_Revenue_Month / MAU`
