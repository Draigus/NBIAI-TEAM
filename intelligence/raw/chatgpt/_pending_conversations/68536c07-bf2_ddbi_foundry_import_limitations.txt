---

### ‚úÖ What DDBI *can* do:
- **Import characters** from D&D Beyond into Foundry, including:
  - Stats, class levels, features, items, spells, etc.
  - Many subclass and racial features
- **Sync character updates** between D&D Beyond and Foundry
- **Import some monsters, items, and spells** that are publicly accessible (e.g., SRD or homebrew)
- Use **PDF-style viewer for sourcebooks** (if hosted legally and configured manually)
- Optionally use **Heroic Roleplaying** integration (a fork with more advanced automation)

---

### ‚ùå What DDBI *cannot* do:
- **Does not import full books** (e.g., *Tasha‚Äôs Cauldron*, *Xanathar‚Äôs Guide*, etc.)
- Cannot bypass D&D Beyond‚Äôs **copyright/DRM restrictions**
- Cannot import **non-SRD content** unless you‚Äôve manually homebrewed it or use *shared access*

Even if you own all the content on D&D Beyond, **the integration only accesses what is legally permitted via the Beyond API or HTML scraping**, which is limited.

---

### Workarounds and Alternatives:
1. **Use the DDB to Foundry module + DDB Importer**  
   This combo works well if you want to import a **character** or **homebrew content** you made on Beyond.

2. **Manually input items or compendiums**

   - Use **homebrew builders** on Beyond to recreate custom items/spells/monsters
   - Use **PDF references** manually
   - Or buy a **Foundry module version** (via the Forge or independent creators like Kobold Press, Ghostfire, etc.)

3. **Use the SRD or community-created content**  
   Many modules come pre-loaded with SRD-equivalent items and spells, which you can modify.

---

### üîÅ How It Works

1. Launches headless Chrome via Puppeteer.
2. Sets your `CobaltSession` cookie for authentication.
3. Navigates to each chapter of the DMG.
4. Extracts the title and text content.
5. Saves each chapter as a Markdown file in `output/DMG`.

---

### üìã The Script

```javascript
// scrape-dmg.js

const fs = require('fs')
const path = require('path')
const puppeteer = require('puppeteer')

const COBALT = process.env.COBALT // your CobaltSession value
if (!COBALT) {
  console.error('‚ùó Set COBALTSession: export COBALT="your-cookie-value"')
  process.exit(1)
}

const BASE_URL = 'https://www.dndbeyond.com'
const BOOK_PATH = '/sources/dnd/dmg-2014'
const OUTPUT_DIR = path.join(__dirname, 'output', 'DMG')

async function main() {
  await fs.promises.mkdir(OUTPUT_DIR, { recursive: true })

  const browser = await puppeteer.launch()
  const page = await browser.newPage()

  // Set your login cookie
  await page.setCookie({
    name: 'CobaltSession',
    value: COBALT,
    domain: '.dndbeyond.com',
    path: '/',
    httpOnly: true,
    secure: true,
  })

  // Visit the DMG main page to find chapter links
  await page.goto(`${BASE_URL}${BOOK_PATH}`, { waitUntil: 'networkidle2' })

  const chapters = await page.$$eval(
    'a.rc-Link--internal[href*="' + BOOK_PATH + '"]',
    links =>
      links.map(link => ({
        title: link.textContent.trim(),
        url: link.href,
      }))
  )

  console.log(`‚û°Ô∏è  Found ${chapters.length} chapters. Beginning scrape...`)

  for (const chap of chapters) {
    console.log(`üìÑ  Scraping: ${chap.title}`)
    await page.goto(chap.url, { waitUntil: 'networkidle2' })

    const content = await page.$eval(
      '.ddbc-content-body',
      el => el.innerHTML
    )

    // Convert HTML to Markdown (simple)
    const md = `# ${chap.title}\n\n` + content
      .replace(/<h2[^>]*>(.*?)<\/h2>/g, '\n## $1\n')
      .replace(/<h3[^>]*>(.*?)<\/h3>/g, '\n### $1\n')
      .replace(/<p[^>]*>(.*?)<\/p>/g, '$1\n\n')
      .replace(/<ul[^>]*>([\s\S]*?)<\/ul>/g, match => {
        return match
          .replace(/<\/?ul[^>]*>/g, '')
          .replace(/<li[^>]*>(.*?)<\/li>/g, '- $1\n')
      })
      .replace(/<strong[^>]*>(.*?)<\/strong>/g, '**$1**')
      .replace(/<em[^>]*>(.*?)<\/em>/g, '_$1_')
      // Remove remaining HTML tags
      .replace(/<[^>]+>/g, '')

    const filename = chap.title
      .toLowerCase()
      .replace(/[^a-z0-9]+/g, '_')
      .replace(/(^_+|_+$)/g, '') + '.md'

    await fs.promises.writeFile(path.join(OUTPUT_DIR, filename), md)
  }

  await browser.close()
  console.log('‚úÖ Done! Markdown files are in', OUTPUT_DIR)
}

main().catch(console.error)
```
