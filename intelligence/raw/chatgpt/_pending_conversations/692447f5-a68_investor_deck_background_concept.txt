3. Variant for dense content slides

For anything with heavy text or financial tables, use a **muted crop** of the same image:

- Take the command deck background, push it 1 to 2 stops darker.  
- Add a soft vertical gradient so the centre is darkest and edges are slightly brighter, which naturally frames the text area.  
- Drop saturation of all colours by about 40 percent so charts and icons pop more.  
- Optionally fade the hologram almost entirely, leaving only faint grid lines and a few ghosted shapes.  

That way you keep brand consistency but preserve readability for numbers and small labels.

# k-means quick via simple random centroid selection and iterations
def kmeans(data, k=4, iters=10):
    centroids = data[np.random.choice(len(data), k, replace=False)]
    for _ in range(iters):
        dists = ((data[:,None,:]-centroids[None,:,:])**2).sum(-1)
        labels = dists.argmin(1)
        new_centroids = np.array([data[labels==i].mean(0) if np.any(labels==i) else centroids[i] for i in range(k)])
        if np.allclose(new_centroids, centroids):
            break
        centroids = new_centroids
    return centroids
