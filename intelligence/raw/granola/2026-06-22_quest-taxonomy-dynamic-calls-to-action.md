# Quest Taxonomy: Replacing "Quest" with Four Dynamic Call-to-Action Types

## Key Content
Framework proposed by a senior game designer (Everwild/Rare background) for structuring player motivation in a live MMO without relying on traditional quest design:

**Why "quest" is a trap:** the word implies "kill five boars," not a real adventure. Fixed quests run out. The game must rely on dynamic and emergent calls to action as its primary content engine.

**Taxonomy (four types):**
1. **Requests/Jobs:** NPC has a problem, player solves it. Dynamic, emergent, won't run out. Reputation consequence, not quest reward.
2. **Challenges:** linear progression tied to a profession or skill (blacksmithing levels, etc.). Player-paced.
3. **Cultural Saga:** chapter-style releases every few months; new mechanic or region; pushes the main narrative. Studio-authored, episodic.
4. **Random encounters/events:** system-produced moments of emergence (wolf attack, ogre ambush, weather event). No authorship required.

**Minecraft principle:** no one gave a quest to build a castle; the world inspired it. Player-set goals last forever; content-fed goals end with "now what?"

**Reputation over quest completion:** doing jobs in a region improves standing, unlocks dialogue options, changes NPC treatment. Not a reward — a consequence.

**Narrative irony:** saving a wolf leads to a child going missing; player feels responsible. Red Dead Redemption 2 cited as best example of weighted consequence chains.

## Decisions / Insights
- Designer concluded: static quest content always runs out; the engagement model must be primarily emergent
- Designer concluded: reputation as a consequence system replaces quest rewards as the engagement hook
- Glen observed: "offer decisions, not A/B choices that converge on the same result"
- Pattern recognised: player-authored goals (Minecraft) produce indefinite engagement; studio-authored goals produce finite engagement with a hard cliff

## Context
Game design session between Glen and a senior game designer advising a ~70-person MMO studio, 2026-06-22. Studio is building a large shared-world live game and designing its core engagement loop.
