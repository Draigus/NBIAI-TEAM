# CH Game Direction: Combat Weapons, Soft Class System, and Player Archetypes

## Key Content

Weekly game direction sync with Robin (Game Director) and Vardis. Dual wielding confirmed as core; stagger system requires visual feedback — stagger intensity matters (milliseconds vs seconds). Every equipped weapon must have unique value: maces for stun/skeletons, axes for bleed/berserker feel, swords for crit/finesse, daggers for high DPS/no stun. Two-handed weapons can stun; one-handed can only stagger. Weapons act as soft class layer: weapon mastery unlocks over time, single weapon first, dual mastery requires reaching skill threshold. Same logic for magic schools. "Weapon as tool" not "weapon as class" — 30–35% of MMO playerbase estimated to be non-combat-primary players. Double RNG loot with weighted system: per-player masking makes tables undatamineable; responsive weighting raises drop chance if player has farmed 500 kills without a target item. Ultra-rare items (~1%) blacklisted from weighting, fixed to specific bosses. Nadir's mandate: design the combat system, not the class system. Business case 2-pager requested: why a gaming hub in Greece.

## Decisions / Insights

- [Glen/Robin/Vardis] decided: weapons are tools not classes; soft class system runs parallel to weapon mastery
- [Glen] decided: 30–35% non-combat archetype estimate means social/crafting classes need equivalent depth
- [Glen/Robin] decided: double RNG + per-player masking for loot — prevents datamining, improves perceived fairness
- [Robin] decided: Nadir's scope = combat system only, not class system design
- [Vardis] initiated: business case for Greek gaming hub at old airport site; Glen to connect with local contacts

## Context

Game direction alignment meeting, 10 June 2026. Glen Pryer, Robin Jubber (Game Director), Vardis (CEO). Recurring weekly design alignment session.
