# Corruption System as PVP/Griefer Containment Honey Pot

## Key Content
Design pattern for managing griefers in a shared-world MMO without hard exclusion:

**Core principle:** griefers are an inevitable player profile; they cannot be eliminated, only channelled. Hard rules shatter the world illusion. Soft incentives and designed outlets work better.

**Corruption system mechanics:**
- Player opts in to "corrupted" role, flips a flag, enters a PVP-enabled zone
- Corrupted jobs: burn houses, extort NPCs; scratches the grief itch in a contained arena
- Transitory design: player can return to PVE after; corruption fades back down
- PVE phase of a region stays clean; PVP phase has its own corruption events
- Creates natural PVP/PVE separation without zone-locking the world

**Historical precedents:**
- EverQuest: killing enough townsfolk made guards hostile on sight; dark elf players had to sneak past their own class trainer — remembered decades later as a powerful immersion memory
- WoW Ratchet: neutral town where attacking guards brought a mob response; felt dynamic rather than punitive

**Design principle:** psychological and social-economic incentives beat hard restrictions. "You don't get to do that" breaks immersion; "you'll face consequences" preserves it.

## Decisions / Insights
- Designer concluded: the corruption flag is a honey pot, not a cage — it makes grief play interesting rather than suppressing it
- Designer concluded: transitory corruption preserves the player's identity; they're not permanently branded
- Glen recalled: EverQuest's faction consequence system as one of his strongest gaming memories — evidence that consequence-based systems create lasting attachment
- Pattern recognised: social-economic incentives (reputation loss, NPC hostility, zone-wide consequences) outperform hard technical locks for community health

## Context
Game design session between Glen and a senior game designer (Everwild/Rare background) advising a ~70-person MMO studio, 2026-06-22. Studio is designing PVP/PVE coexistence for a large shared-world game.
