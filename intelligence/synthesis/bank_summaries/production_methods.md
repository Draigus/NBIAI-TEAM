## Most Recent Additions (June 2026, 12 new extracts)

- **Live service vs box game mindset gap** -- leadership aligned on box game while game requires live service from day one; hard base game cutoff date as the fix; must be raised before producers arrive
- **Milestone as pressure mechanism** -- September set as internal discipline anchor (not conference target); cash runway with no deadline produces drift
- **VS dual purpose** -- investor proof + studio proof it can build; four years of start-stop cycles solved by one complete VS delivery cycle
- **VS real game anxiety** -- "building another pitch deck" pattern; scope anxiety and scope creep are the same phenomenon; fix is explicit mandate, not reassurance
- **VS art quality floor** -- proxy kit + proxy props as VS1 default; mid-poly bake for feature pillars only; artists self-escalate without explicit tier definitions
- **Art Bible as mandatory creative source of truth** -- every art direction change logged in the Bible before communication; style branches within Bible not separate docs; Lore Bible equivalent for narrative
- **Decision owner model** -- named decision owner per pipeline stage; attendees = decision makers only; 12-person/45-minute/no-decisions is the failure pattern
- **Executive RAG meeting format** -- action items first, RAG + plan + date, no problem-solving, persistent item titles week-to-week
- **Poisoned phrase problem** -- terminology from failed initiatives triggers allergic reactions; proactive reframing before others use old language
- **Junior hire policy for remote studios** -- structured training/mentoring/buddy/check-ins mandatory; team shape ~3 veterans, 2 mids, 1 junior
- **Jira setup methodology** -- components over labels (universal); observe a real standup before configuring; two independent admins arrived at the same conclusions
- **Jira parallel-run migration** -- validate parity before cutover; prune backlog during migration; ScriptRunner non-negotiable
