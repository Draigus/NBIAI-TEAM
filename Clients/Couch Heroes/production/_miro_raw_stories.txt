==========================================================================
API TECHNICAL NOTES
==========================================================================

1. Miro's board_list_items API (REST v2) returns org_chart widgets as
   top-level items with empty data. Child cards are NOT enumerated.

2. The context_get API provides AI-summarised content per widget. This
   gives structure and names but NOT raw field data (no exact fill colours,
   no exact hour numbers from card fields). It also sometimes reports
   higher totals ("40+ stories") than it names individually.

3. Individual card fetch via context_get returns: title, note, fill colour,
   border colour, and parent org_chart name. This is the only way to get
   exact card-level data, but requires fetching each card by ID.

4. The 1,621 total items on the board include:
