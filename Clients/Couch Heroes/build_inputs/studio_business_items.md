Six sub-features: GTM and Forecasting, BD, PR, Community, Website, Studio Naming.

Format: `| feature | story | description | status | priority | assignees | health_state | work_type | dependencies | notes |`

Conventions:
- `priority`: critical / high / medium / low
- `status`: open / in_progress / blocked / please_review / resolved
- `health_state`: green / amber / red / unknown
- `work_type`: research / strategy / production / ops / decision / asset / outreach / build
- British English. No timelines. No em dashes.
