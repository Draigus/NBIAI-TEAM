Design Rationale

  • Three-tier hierarchy replaces the current 7-panel scatter. KPI strip for the 5-second scan, table for the 15-second deep read, supporting panels for the 30-second detail pass.
  • Portfolio table is the hero. Every best-in-class PPM tool (Monday, Asana, Smartsheet) centres on a sortable table with one row per client. The current sidebar + 2x2 grid forces the eye to jump between 7 competing zones.
  • Sparklines in KPI cards add trend context. A bare number ("4 needs attention") is less useful than seeing it was 2 last week and trending up. The sparkline communicates direction without requiring the user to read the delta text.
  • RAG dots in a scannable column. The eye processes colour 200ms before text. A vertical column of green/yellow/red dots gives portfolio health in under 2 seconds. The current page uses sidebar cards with coloured left borders, which require 6x more visual space for the same information.
  • Expandable client rows replace the fixed sidebar. Click a client to see their projects inline. This is progressive disclosure: the default view is clean, the detail is one click away.
  • Milestones shown inline as badges in the table, not in a separate panel. This answers "what's coming up for this client?" without leaving the row.
  • "Needs Attention" panel is ranked worst-first. Blocked items above overdue, overdue above at-risk. This is the "what do I act on first?" answer.
  • Timeline below the table, not competing with it. The current page puts the Gantt in a 2x2 grid quadrant where it's too small to be useful. Here it gets full width in a dedicated section.
  • Same CSS variables as WorkSage. Dark and light themes both work. Colours, fonts, spacing are all from the existing design system.
  • Removed: Progress Status donut. A full panel with a backlog bar + donut chart uses enormous space for 5 numbers. Those 5 numbers are now in the KPI strip. Saved ~25% of viewport.
  • Removed: Work Types panel. Low signal-to-noise. "How many active work types" is not actionable. If needed, it could be a filter dropdown.
